📁 File Integrity Monitoring (FIM)
Agregá carpetas para monitorear. El SIEM detectará accesos, modificaciones y eliminaciones de archivos.
No hay carpetas monitoreadas.
¿Cómo funciona? Al agregar una carpeta se aplican reglas SACL en Windows. Se detectan accesos (ID 4663), eliminaciones (4660) y aperturas de handles (4656). Las alertas aparecen clasificadas como HIGH o CRITICAL.
